What is cfc2?

cfc2 is a web app that lets you control your CloudFlare sites via the Client API. Please note that this is a hobby project, and not to be a true replacement for CloudFlare's own dashboard.

Anything else I need to know?

I guess I can iterate here some general non-sense about how user data is stored. Your Client API key is stored as plain text in your browser's Local Storage. Other things stored here includes zone data required for smooth operation of the app. This can be used by a malicious user to mess with your data. If this occurs, this informational page can help you resolve this problem. No user data is stored anywhere on my servers. All data is purely stored in the browser.
